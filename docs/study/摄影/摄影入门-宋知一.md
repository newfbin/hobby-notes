# 01、曝光三要素 [光圈、快门、感光度] & M档调整思路

## 1、光圈

首先找到屏幕上带 "**F**" 的参数，他影响的东西叫光圈。

![image-20251126214010953](./assets/摄影入门-宋知一/image-20251126214010953.png)

光圈就是镜头上面的这个洞，调整 "**F**" 参数就是为了控制这个洞的大小

![image-20251126214542194](./assets/摄影入门-宋知一/image-20251126214542194.png)

需要注意光圈的参数大小和光圈的实际大小是成反比的
